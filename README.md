# Quiz Web App

An interactive quiz web application built using HTML, CSS, and JavaScript. The app allows users to answer multiple-choice questions, move through the quiz, and view their final score at the end.

## Features

- Multiple-choice quiz questions
- Dynamic question and option display
- Option selection
- Score calculation
- Final result display
- Responsive user interface
- Simple and beginner-friendly design

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Highlights

- Used JavaScript to display questions dynamically.
- Implemented DOM manipulation for updating quiz content.
- Added event handling for option selection and button clicks.
- Calculated and displayed the final score based on correct answers.
